## Установка модулей

```
yarn
```

## Запуск в dev-режиме

```
yarn dev
```

Приложение будет доступно по адресу **http://localhost:5173**

## Сборка и предварительный просмотр

```
yarn build
yarn preview
```

Приложение будет доступно по адресы **http://localhost:4173**

## Генерация документации

```
yarn doc
```
